# 🚀 Resume Forge

A modern and interactive resume builder that allows users to create, customize, and export professional resumes with ease.

---

## ✨ Features

* 📄 Live preview of resume
* 🎨 Clean and modern UI
* 📥 Download resume as PDF 
* 🔍 Zoom controls 
* 💾 Import / Export JSON 
* 🖨️ Print to PDF 
* ⚡ Fast and responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
